# Investment-Profit-Comparator
A Java based program that compares the possible future profits from different investment types via looking at the said investments' recent history..
## Overview
The goal of this project is to provide a side-by-side comparison of potential returns from different investment options like Gold, Stocks, and Savings Accounts based on user input.

### Features (In Progress)
* Object-Oriented Programming (OOP): Implementation of core principles such as Encapsulation and Inheritance.
* Profit Logic: Developing algorithms to calculate compound and simple interest rates.
* Data Management: Utilizing Java Collections (e.g., ArrayLists) to manage data.
* File Reading: To store price history and use for predictions.

### Technologies Used
* Language: Java (JDK 17 or higher)
* IDE: Apache NetBeans
